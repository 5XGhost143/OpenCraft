# Tips for first time users like Me

**I highly recommed that u use CLion with CMake and MSVC (used VS26) bc that is what I used you only have to add a few VCPKGs: 1. sdl2 2. glew 3. opengl, that should be all I think. Also to set the vcpkg path go into Settings (CTRL + ALT + S) -> Build, Execution, Deployment -> Click on CMake and select your Preferred Profile (Debug / Release) -> CMake Option: -DCMAKE_TOOLCHAIN_FILE=C:/Users/YourUserName/.vcpkg-clion/vcpkg/scripts/buildsystems/vcpkg.cmake**
